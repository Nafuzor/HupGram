Setup Guide

Подробная Инструкция

01
Авторизация на портале

Перейдите на официальный портал разработчиков my.telegram.org. Введите номер телефона в международном формате. Код придет в Telegram.

02
Выбор раздела

После входа нажмите на "API development tools".

03
Создание приложения

Заполните первые два поля (любым текстом).

App title: HupGramClient
Short name: hgclient

Нажмите синюю кнопку Create application.

04
Копирование ключей

Скопируйте полученные ключи:

App api_id App api_hash

Не передавайте их третьим лицам.

05
Запуск бота

Запустите бота @HupGram_bot и отправьте ему данные.

Support

Частые Вопросы

Могут ли мой аккаунт забанить за юзербота?
Сам факт установки безопасен. Блокировки случаются только при агрессивном использовании модулей: массовый спам в ЛС незнакомцам, рейды. Для администрирования — бана не будет.
Сколько стоит использование HupGram?
Сейчас проект в Beta, поэтому все бесплатно. Позже тяжелые ИИ-модули могут стать платными, но база останется бесплатной для ранних юзеров.
Нужен ли свой сервер (VPS)?
Нет! HupGram базируется на нашем облаке. Установка в 2 клика прямо в Telegram.

1. ОБЩИЕ ПОЛОЖЕНИЯ

  • 1.1. HupGram (далее — "Сервис") — это программный комплекс для автоматизации действий в Telegram (Userbot), предоставляемый на условиях "Как есть" (AS IS).

2. ДАННЫЕ И БЕЗОПАСНОСТЬ

  • 2.1. Для работы Userbot требуется авторизация через ваш Telegram-аккаунт. Вы добровольно предоставляете данные: API ID, API HASH, номер телефона, код авторизации и (при наличии) облачный пароль (2FA).
  • 2.2. Ваши данные используются исключительно для создания сессии и обеспечения работы заявленных функций. Данные хранятся в зашифрованном виде на сервере Администрации.

3. ОБЯЗАННОСТИ ПОЛЬЗОВАТЕЛЯ
